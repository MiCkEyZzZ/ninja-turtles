## Поддержать

Помогите поддержать инфраструктуру Черепашки-ниндзя.
Если вы используете API для своего приложения, онлайн-уроков,
пожалуйста подумайте о том, чтобы поддержать нас, чтобы помочь сохранить проект живым.

Мы не получаем от этого каких-либо денежных средств, и мы используем наше свободное время для поддержания работы API
и обновления данных.
Каждый вклад, каким бы большим или меньшим он ни был, очень ценен для нашего будущего.
Спасибо!
