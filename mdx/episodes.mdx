## Серии
В общей сложности `219` серий отсортированных по `id`

#### Схема серии

| Ключ        | Тип    | Описание                                   |
|-------------|--------|--------------------------------------------|
| id          | Int    | Идентификатор серии                        |,
| name        | String | Наименование серии                         |,
| air_date    | String | Дата выхода серии                          |,
| characters  | Array  | Список персонажей участвующих в серии      |,
| producers   | Array  | Список продюсеров                          |,
| episode     | String | Код серии                                  |,
| season      | String | Код сезона                                 |,
| image_cover | String | Ссылка на изображение эпизода              |,
| url         | String | Ссылка на собственный конечный адрес серии |,
| createdAt   | String | Время создания серии в базе данных         |

#### Получить все серии
Вы можете получить доступ к списку серий, добавив параметр запроса `/episode` в URL-адрес.

```json
{
  "info": {
    "count": 249,
    "pages": 1,
    "next": "https://ninjaturtlesapi.com/api/episode/?page=2",
    "prev": null
  },
  "data": [
    {
      "id": 1,
      "name": "Turtle Trucks",
      "air_date": "December 14, 1987",
      "episode": "01",
      "season": "S01E01",
      "characters": [
        "https://ninjaturtlesapi.com/api/character/1",
        "https://ninjaturtlesapi.com/api/character/2"
      ],
      "producers": [
        "Kara Vallow"
      ],
      "image_cover": "episode-1.png",
      "url": "https://ninjaturtlesapi.com/api/episode/1",
      "createdAt": "2021-04-25T12:42:04.162Z"
    }
  ]
}
```

#### Получить одну серию
Вы можете получить одну серию, добавив  `id`  в качестве параметра: `/episode/28`

```json
{
  "id": 1,
  "name": "Turtle Trucks",
  "air_date": "December 14, 1987",
  "episode": "01",
  "season": "S01E01",
  "characters": [
    "https://ninjaturtlesapi.com/api/character/1",
    "https://ninjaturtlesapi.com/api/character/2"
  ],
  "producers": [
    "Kara Vallow"
  ],
  "image_cover": "episode-1.png",
  "url": "https://ninjaturtlesapi.com/api/episode/1",
  "createdAt": "2021-04-25T12:42:04.162Z"
}
```

#### Фильтрация серий

**Доступные параметры:**

* `name`: фильтрация по заданному имени;
* `season`: фильтрация по заданному коду серии;

```json
{
  "info": {
    "count": 1,
    "page": 1,

    "prev": null
```
