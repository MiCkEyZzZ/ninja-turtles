## Местоположение
Всего существует `15` местоположений отсортированных по `id`.

### Схема местоположения

| Ключ        | Тип    | Описание                                                    |
|-------------|--------|-------------------------------------------------------------|
| id          | Int    | Идентификатор местоположения                                |,
| name        | String | Наименование местоположения                                 |,
| type        | String | Тип местоположения                                          |,
| dimension   | String | Измерение, в котором находится местоположение               |,
| residents   | Array  | Список персонажей, которых в последний раз видели в локации |,
| images      | Array  | Список изображений каждого местоположения                   |,
| image_cover | String | Ссылка на изображение местоположения                        |,
| url         | String | Ссылка на собственную конечную точку местоположения         |,
| createdAt   | String | Время, когда местоположение было создано в базе данных      |

### Получить все местоположения
Вы можете получить доступ к списку местоположений с помощью добавления параметра запроса  `/location` в URL-адрес.

```json
{
  "info": {
    "count": 30,
    "pages": 6,
    "next": "https://ninjaturtlesapi.com/api/location/?page=2",
    "prev": null
  },
  "data": [
    {
      "id": 1,
      "name": "New York City",
      "type": "City",
      "dimension": "unknown",
      "residents": [],
      "images": [],
      "image_cover": "localhost-1.png",
      "url": "https://ninjaturtlesapi.com/api/location/1",
      "createdAt": "2021-04-25T12:42:04.162Z"
    }
  ]
}
```

### Получить местоположение
Для того, чтобы получить одно местоположение вы можете добавить  `id`  в качестве параметра запроса:  `/location/3`

```json
{
  "id": 1,
  "name": "New York City",
  "type": "City",
  "dimension": "unknown",
  "residents": [
    "https://ninjaturtlesapi.com/api/character/1",
    "https://ninjaturtlesapi.com/api/character/2"
  ],
  "images": [],
  "image_cover": "localhost-1.png",
  "url": "https://ninjaturtlesapi.com/api/location/1",
  "createdAt": "2021-04-25T12:42:04.162Z"
}
```

### Получить несколько местоположений
Вы можете получить несколько местоположений, добавив массив `ids` в качестве параметра: `/location/[1,2,3]` или `/location/1,2,3`

```json
[
  {
    "id": 1,
    "name": "New York City",
    "type": "City",
    "dimension": "unknown",
    "residents": [
      "https://ninjaturtlesapi.com/api/character/1",
      "https://ninjaturtlesapi.com/api/character/2"
    ],
    "images": [],
    "image_cover": "localhost-1.png",
    "url": "https://ninjaturtlesapi.com/api/location/1",
    "createdAt": "2021-04-25T12:42:04.162Z"
  },
  {
    "name": "Central Park Zoo",
    "type": "Park",
    "dimension": "unknown",
    "residents": [
      "https://ninjaturtlesapi.com/api/character/12",
      "https://ninjaturtlesapi.com/api/character/21"
    ],
    "images": [],
    "image_cover": "localhost-10.png",
    "url": "https://localhost:8080/api/location/10",
    "createdAt": "2021-04-25T12:42:04.162Z"
  }
]
```

### Фильтрация местоположений

**Доступные параметры:**

* `name`: фильтрация по заданному имени;
* `type`: фильтрация по заданному типу;
* `dimension`: фильтрация по заданному параметру;

```json
{
  "info": {
    "count": 1,
    "pages": 1,
    "next": "https://ninjaturtlesapi.com/api/location/?type=park",
    "prev": null
  },
  "data": [
    {
      "name": "Central Park Zoo",
      "type": "Park",
      "dimension": "unknown",
      "residents": [
        "https://ninjaturtlesapi.com/api/character/12",
        "https://ninjaturtlesapi.com/api/character/21"
      ],
      "images": [],
      "image_cover": "localhost-10.png",
      "url": "https://localhost:8080/api/location/10",
      "createdAt": "2021-04-25T12:42:04.162Z"
    }
  ]
}
```
