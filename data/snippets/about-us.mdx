## О нас

<RestQuery request="GET" endpoint="https://ninjaturtlesapi.com/graphql" />

Привет, меня зовут Михаил!
Я являюсь фанатом The Teenage Mutant Ninja Turtles.
API - это инструмент, который помогает разработчикам во всём мире создавать потрясающие и невероятные веб-сайты и приложения,
используя данные за 30 с лишним лет существования американской франшизы, The Teenage Mutant Ninja Turtles, созданная авторами
комиксов, Кевином Истменом и Питером Лэрдом.

Материалы для заполнения API были получены со следующих ресурсов: [__Википедия__][6], [__Черепахапедия__][7].
В разработке данного API Я использовал [__Node__][0] и typeORM [__Prisma__][1] с базой данной [__PosgresQL__][2].

Все данные отформатированы в формате [__json__][3], и весь проект размещен на [__Digital Ocean__][4] и [__Vercel__][5].
